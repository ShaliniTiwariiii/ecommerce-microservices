# E-Commerce Platform – Microservices Architecture

## Overview
A backend microservices-based e-commerce platform built using Java and Spring Boot.
The system consists of independent services responsible for authentication, product
management, order processing, and payments.

## Microservices
- Auth Service → [https://github.com/ShaliniTiwariii/authJava](https://github.com/ShaliniTiwariii/authJava)
- Product Service → [https://github.com/ShaliniTiwariii/productServices](https://github.com/ShaliniTiwariiii/productServiceMarch26)
- Payment Service →[ https://github.com/ShaliniTiwariii/paymentService](https://github.com/ShaliniTiwariiii/paymentService)

## Tech Stack
- Java, Spring Boot
- Spring MVC, Spring Data JPA
- JWT, OAuth 2.0
- MySQL
- Stripe Payment Gateway
- Redis, Kafka

## Key Features
- Secure authentication and authorization
- RESTful APIs with pagination and exception handling
- Database-per-service architecture
- Third-party payment integration
